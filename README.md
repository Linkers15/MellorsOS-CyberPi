# MellorsOS-CyberPi
A simple OS program for Makeblock CyberPi devices
# How To Install
1. Open mBlock IDE on your computer.
2. Connect your CyberPi via USB or Bluetooth.
3. Switch to Upload Mode
4. In the IDE, select Python, and paste the MellorsOS code into that program.
5. OPTIONAL: If you want to Message other devices over LAN, go to lines 82 and 83 of the program and update your WiFi info.
6. Click upload code and wait for your CyberPi to receive it.
